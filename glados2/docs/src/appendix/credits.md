## Contributions Techniques

Le projet s’appuie sur plusieurs ressources et outils open-source :

- Haskell & Stack — pour la compilation et la gestion du projet
- WebAssembly (WASM) — comme cible de compilation portable
- WABT (WebAssembly Binary Toolkit) — pour le débogage et la conversion du code généré
- Python 3 — pour l’implémentation d’une machine virtuelle bonus (exécution : python vm.py test.wasm)
- mdBook — pour la création de cette documentation
