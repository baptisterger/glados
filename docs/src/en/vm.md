# Virtual Machine (VM)

The **Cédric VM** is a **stack-based virtual machine** that executes the WebAssembly bytecode generated by the compiler.

---

## 1. How It Works

Each operation manipulates an **execution stack**.  
Example:

```ASM
PUSH 3
PUSH 2
ADD
```

Result -> 5 remains on the stack.

---

## 2. Registers and Stack

- Stack: stores intermediate values
- Memory: reserved segment for local variables
- Program Counter (PC): index of the current instruction

---

## 3. Execution Example

Source code:

```cedric
return 2 + 3;
```

Simplified bytecode:

```ASM
PUSH 2
PUSH 3
ADD
RET
```

Execution:

```bash
./virtualmachine test.wasm
```

---

## 4. Python Bonus VM

A bonus Python version allows you to execute .wasm files directly:

```bash
python vm.py test.wasm
```

This VM reads the WebAssembly binary, decodes the instructions, and executes them sequentially.
