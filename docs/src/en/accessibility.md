# Accessibility

The documentation and tools for the Cédric language are designed to be accessible to **people with dyslexia** and anyone who needs enhanced reading comfort.

## Key Features
- High-contrast mode  
- Enlarged text size  
- Wide line spacing  
- Integrated text-to-speech (TTS)  
- Simplified navigation  
- Content available in both english and french

## Accessibility Example in the Documentation
A “Read this page” button is available at the top of each section.  
Click it to listen to the content using your browser’s built-in speech synthesis.
