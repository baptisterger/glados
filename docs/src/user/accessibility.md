# Accessibilité

La documentation et les outils du langage Cédric sont conçus pour être accessibles aux **personnes dyslexiques** et à tout public ayant besoin d’un confort de lecture accru.

## Principales fonctionnalités
- Contraste renforcé
- Taille de texte agrandi
- Interlignage large
- Lecture vocale intégrée (Text-to-Speech)
- Navigation simplifiée
- Contenu disponible en anglais et en français

## Exemple d’accessibilité dans la doc
Un bouton « Lire cette page » est disponible en haut de chaque section.  
Cliquez dessus pour écouter le contenu grâce à la synthèse vocale du navigateur.
